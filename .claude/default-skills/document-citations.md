When Claude Code generates a spreadsheet or word-processing document that incorporates data, statistics, quotes, or other information from external sources, full references for those sources MUST be included.

## Required Behaviour

### Word-processing documents (Word, PDF, Google Docs, etc.)

1. Add a **"References"** section at the end of the document listing every external source cited.
2. Each reference must include enough detail for the reader to locate the original source: title, author or organisation, URL, and access date where applicable.

### Spreadsheets (Excel, CSV, Google Sheets, etc.)

1. Create a separate sheet (tab) named **"References"** rather than appending references at the bottom of a data sheet.
2. List each external source in a clear, tabular format with columns such as: Reference Number, Source Title, Author/Organisation, URL, and Date Accessed.
3. In the data sheets, annotate cells or column headers that rely on external data with a note or comment pointing to the corresponding reference number in the References tab.

### When no external sources are used

If the entire document is generated from user-supplied or internally computed data, no references section or tab is needed.

## When This Skill Applies

- Any spreadsheet or document that Claude Code generates which includes data from external sources.
